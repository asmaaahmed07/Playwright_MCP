# Playwright Automation Project MCP

This project contains automated end-to-end tests for the **OMG web application** using **Playwright** with the **Page Object Model (POM)** design pattern.  
It is built to validate important user flows such as login and project creation while keeping the test structure clean, reusable, and scalable.

---

## Project Overview

The goal of this project is to automate core functional test scenarios of the OMG application, including:

- User login flow
- Project creation flow
- Positive and negative test cases
- Form validation checks
- UI interaction handling
- Stable browser automation using Playwright best practices

This framework is designed to be easy to maintain and extend for future test coverage.

---

## Tech Stack

- **Playwright**
- **JavaScript (Node.js)**
- **Page Object Model (POM)**
- **dotenv** for environment variables
- **GitHub** for version control

---

## Features

- Structured automation framework using POM
- Reusable page classes and methods
- Positive and negative test coverage
- Headed and headless execution support
- Easy configuration using environment variables
- Better maintainability and readability
- Suitable for CI/CD integration in future
